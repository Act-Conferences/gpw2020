Über PerlAbout Perl

Perl ist eine Familie von Open-Source-Programmiersprachen: Perl is a family of Open Source programming languages:

  • Perl 5 wird seit 1987 weiterentwickelt, 2018 wurde die Version 5.28 freigegeben. Perl 5 ist eine der meistgenutzten Programmiersprachen, läuft auf Plattformen vom Handy bis zum Mainframe und bietet mit dem "Comprehensive Perl Archive Network" (CPAN) über 35000 Erweiterungen an. Perl 5 is being developed since 1987, version 5.28 has been released in 2018. Perl 5 is one of the most popular languages and can be used on almost every platform between a mobile phone and a mainframe system. The "Comprehensive Perl Archive Network" (CPAN) offers more than 35.000 extensions.
  • Perl 6 ist eine damit verwandte Sprache, die viele Elemente aktueller Software-Architekturen direkt in die Sprachbeschreibung aufgenommen hat und dafür bewusst auf Kompatibilität mit Perl 5 verzichtet. Perl 6 is a related language which has adopted many paradigms of current software architectures directly into the language, intentionally giving up source compatibility to Perl 5.

Die Perl Foundation ist eine gemeinnützige Organisation, die sich um die Weiterentwicklung und Unterstützung der Gemeinschaft der Benutzer einsetzt. The Perl Foundation is a non-profit organisation dedicated to the advancement of the Perl programmig languages and to supporting the user community.

Perl Mongers sind lokale Benutzergruppen. In Deutschland veranstaltet der Frankfurter Perlmongers e.V. den Deutschen Perl-Workshop, unterstützt von jährlich wechselnden lokalen Organisationen wie den Munich Perl Mongers im Jahr 2019. Perl Mongers are local user groups. In Germany, the Frankfurt Perlmongers e.V. is organizing the annual German Perl Workshop, supported by changing local user groups like the Munich Perl Mongers in 2019.
